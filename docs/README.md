# dotfiles

dotfiles repo managed with [yadm][1].

## Install

`yadm clone git@github.com:TheoBrigitte/dotfiles.git`

## Update

`$HOME/.yadm/bootstrap`

## Sleep

For suspend and hibernation setup using systemd see `docs/sleep/install.sh`.

## Requirements

[yadm][1]
